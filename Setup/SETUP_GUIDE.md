# RelyOn POS: Simple Setup Guide

Follow these 3 easy steps to get the app running on this computer.

## Step 1: Prepare the Database
The app needs a database to store your sales and products. 
1.  Look for the file named `install_mariadb.bat`.
2.  **Right-click** it and select **Run as Administrator**.
3.  Click "Yes" if Windows asks for permission. 
4.  Wait for the black window to finish and close. 
    *(Make sure you have an internet connection for this part).*

---

## Step 2: Install the App
1.  Find the installer file: `RelyOn POS_0.1.0_x64_en-US.msi` (or the `.exe`).
2.  Double-click it and follow the steps to install the app.

---

## Step 3: Open the App
1.  Go to your **Start Menu** and search for **RelyOn POS**.
2.  Open it. Everything starts automatically!

---

### Forgot your login?

### Need Help?
- **Is there an error?** Try right-clicking `install_mariadb.bat` and choosing **Run as Administrator** again.
- **Still not working?** Check if your internet is on during the first step.
